# ADB 无线配对使用指南

## 🚀 快速开始

### 使用脚本（推荐）

```bash
python adb_pair_correct.py
```

选择"1"进行配对，按照提示操作即可。

### 手动命令行方式

```bash
# 1. 在手机上操作
# - 打开"设置" > "开发者选项" > "无线调试"
# - 点击"使用配对码配对设备"
# - 记下手机显示的IP、端口和配对码

# 2. 在Mac上执行配对（替换为手机显示的信息）
adb pair 192.168.2.xxx:xxxxx

# 3. 输入手机上显示的6位配对码，按回车

# 4. 配对成功后连接设备
adb connect 192.168.2.xxx:5555

# 5. 验证连接
adb devices
```

## 📱 注意事项

1. **同一网络**: 确保手机和Mac在同一WiFi
2. **Android版本**: 需要Android 11或更高
3. **配对方向**: 配对码由**手机生成**，电脑输入

## 🔧 常见问题

### 连接端口不是5555？

在手机的"无线调试"页面查看"IP地址和端口"，使用显示的端口号连接。

### 已配对过，想快速重连？

```bash
python adb_pair_correct.py
```

选择"2"快速连接模式。
